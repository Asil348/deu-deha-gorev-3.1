## DEU DEHA GÖREV 3

### Kurulum

---

1. Projeyi klonlayın.
1. mysql veritabanı oluşturun. Veritabanı konfigurasyonunu `server/index.js` dosyasından tanımlayın.
1. Sunucuları çalıştırın:

#### Frontend sunucusu için:
```
cd client/
yarn
yarn dev
```

build için:
```
yarn build
```

#### Backend sunucusu için:
```
cd server/
npm i
npm run start
```
